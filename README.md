# Netlify SSR Form

```bash
npm init svelte@next sveltekitnetlifyforms
cd sveltekitnetlifyforms
npm install
git init
npm i @sveltejs/adapter-netlify@next # & add to: `svelte.config.js`
npm run dev -- --open
```

Navigate to [localhost:3000](http://localhost:3000).
